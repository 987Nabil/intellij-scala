Reports usage of the ? identifier in kind-projector syntax.

Teh ? identifier is going to be used for wildcard arguments in types in Scala 3 (see Wildcard Arguments in Types). Its usage in kind-projector type lambdas is deprecated and is going to be superseeded by *.

Example:


  List[?]

After the quick-fix is applied:


  List[*]

This inspection runs when the Kind Projector plugin is enabled.
