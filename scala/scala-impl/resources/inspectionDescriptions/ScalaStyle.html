Reports problems found by Scalastyle.

Scalastyle examines your Scala code and indicates potential problems with it.

Place the config file in <project-root>, <project-root>/.idea, or <project-root>/project.

The name of the config file can be scalastyle-config.xml or scalastyle_config.xml. Config files for test sources can be overridden by scalastyle-test-config.xml or scalastyle_test_config.xml.

The full documentation is available on the Scalastyle website.
